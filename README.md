# Okio Zstd

Build the native libraries:

# Git Submodules

```
git submodule init
git submodule update
```

# cmake

```
brew install cmake
```

# macOS

```
./okio-zstd/src/jvmMain/build-mac.sh
```

# Linux

```
./okio-zstd/src/jvmMain/build-linux.sh
```

# Test it:

```
./gradlew okio-zstd:jvmTest
```
